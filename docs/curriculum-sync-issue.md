## 3. 향후 커리큘럼 방향 아이디어 (논의됨, 확정 아님)

아루나가 제안한 23기 이후 커리큘럼 개편 방향:

| 주차 | 내용 |
|---|---|
| 1주차 | 툴 세팅 (VS Code, Claude Code, bkit, Node.js, GitHub 가입) |
| 2주차 | AI 메타학습 (막혔을 때 물어보기, CLAUDE.md, 계획서 받고 진행) |
| 3주차 | bkit으로 랜딩페이지 만들기 (아무것도 없는 상태에서 직접 만들기) |
| 4주차 | 내가 준 레포 가져와서 응용 + 심화: 다른 재밌는 레포 탐험 |

**이 방향의 장점:**
- 메타스킬 트랙이 커리큘럼 안에 자연스럽게 위치
- "레포 가져오기만 하는 것"이 아니라 "직접 만들어보기"도 포함
- 나중에 더 큰 것 만들 때 기반이 되는 능력 훈련

**보류 이유**: 22기 진행 중이라 지금 바꾸지 않음. 23기 기획 시 재논의.
