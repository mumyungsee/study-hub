# /mission — 현재 미션 확인

## 역할

수강생이 현재 진행할 미션을 확인하고 시작을 돕습니다.

## 실행 순서

1. `00_missions/` 폴더에서 이미 제출한 파일(`mission-*.md`, `template/` 제외) 있는지 확인
2. 있으면 → "이미 제출 준비된 파일이 있네요. `/submit`으로 제출하면 됩니다" 안내
3. 없으면 → 현재 미션 내용 안내

## 미션 안내 형식 (파일 없을 때)

```
현재 미션: 미션 1 — 나만의 라이프 위키 만들기

완료할 것:
- Fork 완료
- Vercel 배포 완료 (내 URL 생김)
- 사이트에 내 이름/소개 반영

관련 챕터: 1~6장

준비됐으면 "미션 제출해줘" 또는 /submit 을 입력하세요.
파일 작성부터 PR 제출까지 제가 다 도와드립니다.
```
