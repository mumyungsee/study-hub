# /mission — 현재 미션 확인

## 역할

수강생이 현재 진행할 미션을 확인하고 제출 방법을 안내합니다.

## 실행 순서

1. `00_missions/README.md`를 읽어서 미션 목록 파악
2. `00_missions/` 폴더에서 수강생이 이미 제출한 파일이 있는지 확인
3. 미완료 미션을 찾아서 안내

## 출력 형식

```
현재 미션: 미션 N — [제목]

할 일:
1. 00_missions/template/mission-0N.md 파일을 복사합니다
2. 파일명을 mission-0N-[내GitHub아이디].md 로 바꿉니다
3. 파일을 열어서 빈 칸을 채웁니다
4. 완료되면 `/submit`을 입력하세요

관련 챕터: N~M장
```

## 주의사항

- 미션 파일이 이미 있으면 "이미 제출 준비가 됐네요. `/submit`으로 제출하면 됩니다" 안내
- 모든 미션 완료 시 갤러리 등록 안내 (`/gallery`)
