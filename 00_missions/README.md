# 미션 제출 방법

미션을 완료하면 AI한테 "미션 제출해줘"라고 하세요. AI가 PR을 대신 만들어드립니다.

## 순서

1. `00_missions/template/` 폴더에서 해당 미션 파일을 복사합니다
2. 파일명에서 `template/`를 빼고 내 GitHub 아이디를 붙입니다
   - 예: `mission-01-myname.md`
3. 파일을 열어서 빈 칸을 채웁니다
4. 채팅창에 `/submit` 또는 "미션 제출해줘"라고 입력합니다
5. AI가 PR을 생성합니다 — 강사가 확인 후 승인합니다

## 미션 목록

| 미션 | 내용 | 관련 챕터 |
|------|------|---------|
| 미션 1 | 나만의 라이프 위키 만들기 | 1~6장 |

## 제출 전 확인

- 파일명에 내 GitHub 아이디가 들어가 있나요?
- 배포 URL을 적었나요?
- 체크리스트를 완료했나요?
