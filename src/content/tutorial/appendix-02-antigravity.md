## Antigravity가 뭔가요

Antigravity는 Google이 만든 AI 코딩 IDE입니다. VS Code(Visual Studio Code)를 기반으로 만들어졌습니다. 왼쪽에 파일 탐색기, 가운데에 에디터, 오른쪽에 AI 채팅창이 붙어있는 구조입니다.

비개발자 관점에서 쉽게 말하면, **"AI 비서가 옆에 붙어있는 워드 프로세서"**입니다. 일반 문서 편집기처럼 파일을 열고 수정하는데, 오른쪽 채팅창에서 AI에게 말을 걸면 파일을 직접 수정해줍니다.

---

## 화면 구성

```
┌──────────┬─────────────────────┬──────────────┐
│ 파일     │                     │              │
│ 탐색기   │    에디터           │  AI 채팅창   │
│ (왼쪽)   │    (가운데)         │  (오른쪽)    │
│          │                     │              │
├──────────┴─────────────────────┴──────────────┤
│                  터미널 (아래)                  │
└────────────────────────────────────────────────┘
```

### 파일 탐색기 (왼쪽)

열어놓은 폴더의 파일 목록이 표시됩니다. 파일을 클릭하면 가운데 에디터에서 열립니다.

보이지 않으면 `Ctrl+Shift+E`를 누르거나, 왼쪽 사이드바의 파일 두 장 겹친 모양 아이콘을 누릅니다.

### 에디터 (가운데)

파일 내용을 직접 보고 수정하는 공간입니다. 여러 파일을 탭으로 열 수 있습니다.

### AI 채팅창 (오른쪽)

Claude와 대화하는 곳입니다. 이 튜토리얼의 모든 요청은 여기서 합니다.

`Ctrl+L`을 누르면 채팅창이 열립니다. 채팅창이 안 보이면 이 단축키를 먼저 눌러보세요.

### 터미널 (아래)

Claude가 명령어를 실행할 때 결과가 표시되는 곳입니다. 직접 명령어를 입력할 수도 있습니다.

`Ctrl+\``(물결표 왼쪽 키)로 열고 닫습니다.

## 핵심 단축키

외울 필요는 없습니다. 막힐 때 여기서 찾으세요.

### 자주 쓰는 것

| 단축키 | 기능 |
|--------|------|
| `Ctrl+L` | AI 채팅창 열기 |
| `Ctrl+E` | Agent Manager 뷰 열기 (전체화면 에이전트) |
| `Ctrl+I` | 에디터에서 인라인 명령 (코드 일부 수정 요청) |
| `Ctrl+Shift+P` | 명령 팔레트 (기능 검색) |

### 화면 조정

| 단축키 | 기능 |
|--------|------|
| `Ctrl+B` | 왼쪽 파일 탐색기 접기/펴기 |
| `` Ctrl+` `` | 터미널 열기/닫기 |
| `Ctrl+,` | 설정 열기 |

## Agent Manager 뷰

`Ctrl+E`를 누르면 에이전트 전용 전체화면 뷰로 전환됩니다. 코드 에디터가 사라지고 에이전트 대화창이 큰 화면에서 열립니다.

복잡한 작업을 지시할 때 유용합니다. 에디터가 없어도 되는 상황, 긴 작업을 맡길 때 이 화면에서 합니다.

`Ctrl+E`를 다시 누르면 원래 에디터 화면으로 돌아옵니다.

---

## 인라인 명령 (Ctrl+I)

에디터에서 파일을 열어두고, 수정하고 싶은 부분을 드래그로 선택한 다음 `Ctrl+I`를 누르면 작은 입력창이 뜹니다.

```txt
이 부분 더 간결하게 고쳐줘.
```

오른쪽 채팅창까지 가지 않고 에디터 안에서 바로 요청하는 방법입니다.

## 막힐 때

### 채팅창이 안 보일 때

`Ctrl+L`을 눌러보세요.

### 터미널이 안 보일 때

`` Ctrl+` ``를 눌러보세요.

### 메뉴를 못 찾겠을 때

`Ctrl+Shift+P` → 검색창에 원하는 기능 이름을 입력하면 됩니다. 예: `terminal`, `settings`, `extensions`.

### 화면이 너무 복잡할 때

`Ctrl+B`로 왼쪽 탐색기를 접으면 에디터와 채팅창만 남습니다.
