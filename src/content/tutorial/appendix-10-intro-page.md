VS Code와 Claude Code가 이미 깔려 있는 분들을 위한 보너스 실습입니다. 약 45분 정도 걸리고, 결과물도 손에 쥐고 가는 시간입니다.

## 💡 이 실습의 진짜 학습 포인트

아래 안내는 **"이런 식으로 자연어로 말하면 된다"는 가이드일 뿐**입니다. 똑같이 따라 말하지 마세요. 본인의 말투, 본인의 요구사항으로 **자유롭게 응용**해보세요.

복사·붙여넣기로 끝내면 다음번에 또 못 합니다. **AI한테 자연어로 말하는 감각이 이 실습의 진짜 결과물**입니다.

(혹시 막막하면 각 단계 아래 `참고용 완성 예시 프롬프트`를 펼쳐서 참고하세요. 다만 가능하면 본인 말로 먼저 시도해보세요.)

## 이 실습이 노리는 것

단순히 페이지 하나 만드는 게 아닙니다.

1. **CLAUDE.md**를 통해 AI에게 일하는 방식(PDCA 라이브코딩)을 가르치고
2. AI가 인터뷰로 정보를 끌어내면서 **미니 링크트리(자기소개 페이지)**를 같이 만들고
3. 그 흐름을 **스킬로 저장**해서 다음에 한 줄로 재사용 가능하게 합니다

## 준비 — 작업 폴더 만들기

새 폴더로 시작하는 경우 (PowerShell):

```powershell
mkdir C:\dev\my-intro; cd C:\dev\my-intro; claude
```

## STEP 1. CLAUDE.md를 학습 가이드로 만들기

AI한테 **"앞으로 이 폴더에서 일할 때는 이렇게 행동해라"**를 가르치는 작업입니다.

### 이런 식으로 자연어로 말하면 됩니다

> *"학습 가이드로 쓸 CLAUDE.md 파일을 만들고 싶어. 이런 식이었으면 좋겠어:*
>
> - *AI는 강사가 아니라 학습 메이트처럼 행동했으면 좋겠어*
> - *모르는 거 있으면 먼저 물어보고, 같이 풀어가는 식으로*
> - *어려운 건 쉽게 풀어서 설명해줘*
> - *작업할 때는 PDCA 순서로 — 계획 세우고, 투두 만들고, 실행하고, 점검하고, 개선하고*
> - *디자인 같이 정해야 할 때는 일일이 물어보지 말고 트렌디한 스타일 몇 개 선택지로 보여주기*
>
> *이 내용으로 CLAUDE.md 만들어줘."*

### 본인이 응용할 때 넣을 만한 요구사항 예시

- AI 태도: 강사 / 학습 메이트 / 친구 / 비서 등 어떤 톤이 좋은지
- 설명 방식: 어려운 용어 풀어쓰기 / 비유 많이 / 짧고 직설적 등
- 작업 방식: PDCA / 한 번에 끝내기 / 천천히 단계별
- 인터뷰 스타일: 객관식 선택지 / 한 번에 다 묻기 / 한 개씩 묻기
- 결과물 보여주는 방식: 코드만 / 미리보기 같이 / 설명 길게

**자기 스타일대로 자유롭게 더하거나 빼세요.** 본인이 어떤 식으로 학습하고 싶은지가 가장 중요합니다.

<summary>참고용 완성 예시 프롬프트 (가능하면 본인 말로 먼저 시도해보세요)</summary>

```text
이 폴더에 CLAUDE.md 파일을 만들어줘. 이 파일은 앞으로 이 폴더에서

이런 식이었으면 좋겠어:

- 너는 강사가 아니라 학습 메이트처럼 행동해줘
- 모르는 거 있으면 먼저 물어보고, 같이 풀어가는 식으로 진행해줘
- 어려운 건 쉽게 풀어서 설명해줘 (비개발자도 이해할 수 있게)
- 작업은 PDCA 사이클로 진행해줘:
  1. Plan: 사용자가 원하는 걸 다시 확인하고 큰 그림 잡기
  2. Do: 작은 투두로 쪼개서 진행 상황 보이며 작업
  3. Check: 결과 보여주고 피드백 받기
  4. Act: 피드백 받은 부분만 정확히 수정
- 디자인 같이 정해야 할 때는 일일이 물어보지 말고
  트렌디한 스타일 몇 개를 선택지로 보여주는 방식으로 인터뷰해줘

이 요구사항으로 CLAUDE.md 작성하고, 다 됐으면 파일 내용 보여줘.
```
</details>

파일이 만들어지면 한 번 열어보세요. **"AI한테 일하는 방식을 가르치는 게 가능하다"**는 걸 눈으로 확인하는 게 중요해요.

## STEP 2. 미니 링크트리 만들기 — AI와 PDCA 협업

이제 본격 작업입니다. CLAUDE.md를 만들었으니 AI가 알아서 Plan부터 시작합니다.

### 이렇게 짧게만 말해도 충분합니다

> *"미니 링크트리(자기소개 페이지) 만들고 싶어.*
> *모바일 비율, 한 파일짜리 HTML로. 인터뷰부터 시작해줘."*

CLAUDE.md에 PDCA 방식이 적혀 있으니, AI가 **자기소개 인터뷰**와 **디자인 인터뷰**를 알아서 시작할 거예요.

### 인터뷰에서 답하게 될 내용

- **자기소개:** 이름·닉네임, 한 줄 소개, 인스타·링크드인·이메일·블로그 등 연결할 곳
- **디자인:** AI가 트렌디한 스타일 몇 개를 선택지로 보여줄 거예요. 그중에서 마음에 드는 것 고르거나 *"이 스타일 비슷한데 살짝 더 부드럽게"* 식으로 응용해서 답하면 됩니다.

> **스크린샷 활용 팁** — 마음에 드는 디자인이 있으면 스크린샷을 폴더에 넣고 *"이런 느낌으로 만들어줘"* 라고 알려주세요. 인스타그램 프로필, 마음에 드는 웹사이트 등 뭐든 좋아요. Claude Code는 폴더 안 이미지를 읽을 수 있어요.

### 결과 확인 & 수정 (Check → Act)

인터뷰가 끝나면 AI가 `index.html`을 생성합니다. 더블클릭하면 브라우저에서 결과가 떠요.

마음에 안 드는 부분이 있으면 **그냥 자연어로 말하세요**. AI가 즉시 반영합니다.

> *"색이 너무 진해, 좀 더 부드럽게."*
> *"버튼 간격 조금 더 넓게."*
> *"아바타를 이모지 말고 이니셜로 바꿔줘."*

만족스러울 때까지 한두 번 반복하면 됩니다.

```text
미니 링크트리(자기소개 페이지)를 index.html 한 개 파일로 만들고 싶어.

참고할 스크린샷이 있으면 폴더에 넣어달라고 안내해줘.
```
</details>

## STEP 3. 이 흐름을 스킬로 저장하기

방금 만든 흐름이 마음에 든다면, **재사용 가능한 스킬**로 저장합시다. 다음번엔 한 줄 호출로 똑같이 만들 수 있어요.

### 이 정도 자연어로 충분합니다

> *"방금 우리가 한 작업 — 자기소개 인터뷰부터 미니 링크트리 만들기까지 — 을*
> *다음에도 또 쓸 수 있게 스킬로 저장해줘. 이름은 `intro-page`로 하고,*
> *다음에 어떻게 호출하는지도 알려줘."*

본인이 강조하고 싶은 디테일이 있으면 자연어로 덧붙이세요. 예: *"인터뷰는 한 번에 하나씩 묻게"*, *"디자인 스타일 선택지는 꼭 보여주게"* 등.

<details>
<summary>참고용 완성 예시 프롬프트 (가능하면 본인 말로 먼저 시도해보세요)</summary>

```text
방금 우리가 한 작업 — "자기소개 인터뷰 → 디자인 인터뷰 →
미니 링크트리 index.html 생성" 흐름 — 을 Claude Code 스킬로 저장해줘.

- 스킬 이름: intro-page
- 저장 위치: .claude/skills/intro-page/SKILL.md
- 호출 시 동작:
  1. 사용자에게 자기소개 인터뷰 (이름·소개·SNS 링크, 한 번에 하나씩)
  2. 디자인 인터뷰 (트렌디한 스타일 선택지 보여주기, 참고 스크린샷 허용)
  3. index.html 미니 링크트리 페이지 생성 (모바일 비율, inline CSS,
     Google Fonts 한글 폰트, 그림자·hover 효과)
  4. 브라우저로 여는 방법 안내

만든 후 어떻게 호출하는지 사용법도 알려줘.
```
</details>

## STEP 4. (선택) 스킬 테스트 — "진짜 한 줄로 되나?"

새 폴더를 만들고 스킬을 호출해서 친구·가족용 페이지를 즉석 생성해보세요.

```text
/intro-page 호출해서 진행해줘.
```

스킬이 잘 만들어졌다면 다시 인터뷰부터 시작해서 새 페이지를 만들어줄 거예요. **"오, 진짜 한 줄로 되네"**를 체험하는 순간이에요.

## 이 실습이 가르치는 것

1. **CLAUDE.md = AI에게 일하는 방식을 가르치는 지시서** — 도구를 외우는 게 아니라 AI의 행동을 정의합니다
2. **AI한테 자연어로 말하는 감각** — 완성된 프롬프트를 외우는 게 아니라 본인 말투로 요구사항 전달하는 법
3. **AI는 도구가 아니라 학습 메이트** — 먼저 묻고, 디자인 인터뷰도 같이 하고, PDCA로 협업합니다
4. **잘 풀린 작업은 스킬로 자산화** — 한 번 만든 흐름은 평생 한 줄로 재사용됩니다

이게 매뉴얼 따라하기가 아니라 **AI와 함께 학습하는 메타학습**의 실제 모습입니다.
